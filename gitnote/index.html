プライバシーポリシー - GitNote

最終更新日: 2026年3月21日

Hiroyuki Takeda（以下「当方」）は、GitNote（以下「本サービス」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。

1. 収集する情報

1.1 GitHub認証を通じて取得する情報

本サービスでは、GitHub OAuth認証を利用してログインします。認証時に以下の情報を取得します。

  • GitHubアカウント名（ユーザー名）
  • メールアドレス
  • プロフィール画像URL

1.2 サービス利用に伴う情報

本サービスの機能提供のため、以下の情報を使用します。

  • 接続先GitHubリポジトリ名・ブランチ名
  • ノートのメタデータ（タイトル、タグ、作成日時、更新日時）

2. データの保存場所

  • ノートデータ: ユーザー自身のGitHubリポジトリに直接保存されます。当方のサーバーにノート本文を保存することはありません。
  • セッション情報: 認証セッションの管理のためサーバー上に保存されます。
  • ローカルストレージ: UI設定（テーマ、サイドバー状態等）はブラウザのローカルストレージに保存されます。

3. 情報の利用目的

  • ユーザー認証およびセッション管理
  • GitHubリポジトリとのノート同期機能の提供
  • サービスの改善および不具合の修正

4. 第三者サービス

本サービスでは以下の第三者サービスを利用しています。

GitHub（GitHub, Inc.）

  • 目的: ユーザー認証、ノートデータの保存・同期
  • プライバシーポリシー: GitHub プライバシーステートメント

Vercel（Vercel Inc.）

  • 目的: Webアプリケーションのホスティング
  • プライバシーポリシー: Vercel プライバシーポリシー

5. ユーザーの権利

  • データへのアクセス: ノートデータはユーザー自身のGitHubリポジトリに保存されているため、いつでもアクセス・管理できます。
  • アカウント連携の解除: GitHubの設定からOAuth連携を解除できます。
  • データの削除: ログアウトによりセッション情報が削除されます。ブラウザのローカルストレージはブラウザの設定から削除できます。

6. Cookie・ローカルストレージの使用

  • セッションCookie: ログイン状態の維持に使用します。
  • ローカルストレージ: テーマ設定、サイドバーの開閉状態等のUI設定の保持に使用します。

7. 子どものプライバシー

本サービスは13歳未満のお子様を対象としていません。

8. プライバシーポリシーの変更

当方は、必要に応じて本プライバシーポリシーを変更することがあります。変更があった場合は、本ページを更新いたします。
