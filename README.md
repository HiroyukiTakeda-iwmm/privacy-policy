# iWMM Works - プライバシーポリシー

iWMM Works が提供するアプリケーション用のプライバシーポリシーページです。
GitHub Pages でホスティングしています。

## 公開URL

| アプリ | URL |
|--------|-----|
| トップページ | https://hiroyukitakeda-iwmm.github.io/privacy-policy/ |
| Slot Analyzer | https://hiroyukitakeda-iwmm.github.io/privacy-policy/slot-analyzer/ |
| Pachi Manager | https://hiroyukitakeda-iwmm.github.io/privacy-policy/pachi-manager/ |

## ファイル構成

```
privacy-policy/
├── index.html              # トップページ（アプリ一覧）
├── assets/
│   └── style.css           # 共通スタイル
├── slot-analyzer/
│   └── index.html          # Slot Analyzer プライバシーポリシー
├── pachi-manager/
│   └── index.html          # Pachi Manager プライバシーポリシー
└── README.md
```

## 新しいアプリの追加手順

1. アプリ名のディレクトリを作成（例: `new-app/`）
2. 既存の `index.html` をコピー
3. アプリ固有の部分を修正（アプリ名、収集データの説明）
4. ルートの `index.html` にリンクを追加
5. コミット & プッシュ
