プライバシーポリシー - Pachi Manager

最終更新日: 2026年4月1日

Hiroyuki Takeda（以下「当方」）は、Pachi Manager（以下「本アプリ」）におけるユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーでは、本アプリにおける情報の取扱いについて説明します。

1. 収集する情報

本アプリでは、収支管理に関するデータ（収支記録、ホール情報等）をデバイス上にローカル保存します。これらのデータはデバイス外に送信されることはありません。

本アプリは外部サーバーとの通信を行わず、ユーザーの個人情報を収集・送信することはありません。

2. カメラ・写真ライブラリのアクセスについて

本アプリでは、収支記録に画像を添付する機能を提供しています。この機能を利用する際、カメラまたは写真ライブラリへのアクセス許可を求める場合があります。

  • カメラ: 収支記録に添付する画像を撮影するために使用します。
  • 写真ライブラリ: 収支記録に添付する画像を選択するために使用します。

撮影・選択された画像はデバイス内にのみ保存され、外部に送信されることはありません。

3. 第三者への情報提供

本アプリでは第三者サービスを利用しておらず、ユーザーデータを外部に提供・送信することはありません。

4. データの保存と管理

すべてのデータはユーザーのデバイス内にのみ保存されます。当方がユーザーのデータにアクセスすることはできません。

5. ユーザーの権利

  • データの削除: アプリ内の設定画面から全データを削除できます。また、アプリをアンインストールすることで、デバイス上のすべてのアプリデータが削除されます。
  • データのエクスポート: アプリ内の設定画面からデータをJSON形式でエクスポートできます。

6. 子どものプライバシー

本アプリは13歳未満のお子様を対象としていません。13歳未満のお子様から意図的に個人情報を収集することはありません。

7. プライバシーポリシーの変更

当方は、必要に応じて本プライバシーポリシーを変更することがあります。変更があった場合は、本ページを更新いたします。
